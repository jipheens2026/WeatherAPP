## Overview
This project is a React/Vite dashboard that integrates Weather-AI's API to deliver:

- Interactive charts for temperature and precipitation trends
- Image-based tree canopy analysis and farm health insights

- `README.md` — Setup, deployment, and project overview
- `netlify.toml` — Netlify deployment config
- `.env.example` — Environment variable template
- `src/api/weatherAPI.js` — Weather-AI API client and endpoints
- `src/components/WeatherDashboard.jsx` — Weather UI and custom coordinate search
- `src/components/TreeAnalysis.jsx` — Tree analysis upload flow

## How to Run Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` from `.env.example`
4. Add your Weather-AI key:
   ```bash
   VITE_WEATHER_AI_API_KEY=wai_your_actual_api_key_here
   ```
5. Start the app: `npm run dev`
